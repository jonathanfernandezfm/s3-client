# S3Dock

S3Dock is a web application for managing S3-compatible object storage. Connect to multiple S3 endpoints (AWS S3, MinIO, etc.), browse buckets, upload/download files, and manage objects.

## MCP Server (AI assistant integration)

S3Dock includes a built-in [Model Context Protocol](https://modelcontextprotocol.io/) server that lets AI assistants (Claude Desktop, Cursor, etc.) browse your S3 connections and generate download links.

See **[docs/mcp.md](docs/mcp.md)** for setup instructions.
